GetTranslator

Get a function that can translate strings for the current language

Translate:GetTranslator()

Returns

ta function which takes a single token in string form, and returns the translated string.  If the token isn’t defined, it returns the token.

Example

local t = Translate:GetTranslator()
UI.Message(t('YOU_ARE_LATE'))

Availability

alpha 15

Status

experimental
