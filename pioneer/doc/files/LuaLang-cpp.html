GetDictionary

Retrieve a Lua table for the current language.

dict = Lang.GetDictionary()
print(dict['WE_HAVE_NO_BUSINESS_WITH_YOU'])

GetCoreLanguages

Retrieve a Lua table listing the languages for which core translations are available.
